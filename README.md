# Projet Flux de Données et Accès Concurrents
Le principe est d'effectuer un compteur de mots à la map-reduce, le tout sans utiliser la platforme Hadoop prenant :
 - En entrée : un ensemble de fichiers texte contenant des mots
 - En sortie : un dictionnaire comportant l’ensemble des mots associés au compteur de leur nombre d’occurrences

---
## Getting Started
Projet trouvable sur GitHub au lien suivant : https://github.com/Jopruvo/projet_flux

---
## Usage
Excécutez le fichier "Main.java" de votre choix selon si vous voulez la version avec 2 mappers ou *n* mappers
